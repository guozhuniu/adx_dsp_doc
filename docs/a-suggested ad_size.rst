建议广告尺寸
==========================

1.banner推荐规格
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
320*50、468*60、728*90、640*100

2.插屏推荐规格
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
* 横屏
   普通：300*250、标清：600*500、高清：960*640  
* 竖屏
   普通：250*300、标清：500*600、高清：640*960

3.开屏推荐规格
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
640*960、720*1080

4.视频推荐规格
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
* video
   视频格式：MP4
   视频时长：15s、30s
   视频大小：5M以内
* 展示图片
   物料格式：图片
   尺寸：
      横屏：960*640、竖屏：640*960
      
5.原生推荐规格
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~     
图片尺寸比例：1:1~1：2之间
